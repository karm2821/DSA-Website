Introduction to Hashing

Hashing is a technique used to uniquely map data to a fixed-size value, called a hash, using a hash function. It is widely used in data structures like hash tables and cryptographic applications.

Types of Hashing

  • Direct Hashing: Uses the key itself as the index.
  • Modulo Hashing: Uses key % table_size to determine the index.
  • Collision Resolution: Techniques like chaining and open addressing to handle hash collisions.

Hashing Examples

Python

hash_table = {}
hash_table["name"] = "Alice"
print(hash_table["name"])  # Output: Alice

Java

import java.util.HashMap;
HashMap<String, String> hashMap = new HashMap<>();
hashMap.put("name", "Alice");
System.out.println(hashMap.get("name")); // Output: Alice

C++

#include <unordered_map>
std::unordered_map<std::string, std::string> hashMap;
hashMap["name"] = "Alice";
std::cout << hashMap["name"]; // Output: Alice
