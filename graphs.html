Introduction to Graphs

A graph is a data structure that consists of a set of nodes (vertices) and a set of edges that connect these nodes. Graphs can be used to represent networks, relationships, and more.

Types of Graphs

  • Directed Graph: Edges have a direction from one vertex to another.
  • Undirected Graph: Edges do not have a direction.
  • Weighted Graph: Edges have weights associated with them.
  • Unweighted Graph: Edges do not have weights.

Graph Examples

Python

graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}
print(graph['A'])  # Output: ['B', 'C']

Java

import java.util.*;
Map> graph = new HashMap<>();
graph.put("A", Arrays.asList("B", "C"));
graph.put("B", Arrays.asList("A", "D", "E"));
graph.put("C", Arrays.asList("A", "F"));
System.out.println(graph.get("A")); // Output: [B, C]
