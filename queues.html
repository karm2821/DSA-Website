Introduction to Queues

A queue is a linear data structure that follows the First In, First Out (FIFO) principle. Elements are added at the rear and removed from the front of the queue.

Types of Queues

  • Enqueue: Adds an element to the rear of the queue.
  • Dequeue: Removes the front element from the queue.
  • is Empty: check queue is empty or not.

Queues Examples

Python

from collections import deque
queue = deque()
queue.append(1)  # Enqueue
queue.append(2)
print(queue.popleft())  # Dequeue: Output 1

Java

import java.util.LinkedList;
import java.util.Queue;
Queue queue = new LinkedList<>();
queue.add(1);
queue.add(2);
System.out.println(queue.poll()); // Output: 1

C++

#include 
std::queue queue;
queue.push(1);
queue.push(2);
std::cout << queue.front(); // Output: 1
queue.pop();
